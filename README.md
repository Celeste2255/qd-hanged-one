    *** Descrizione del progetto ***

Creazione di un gioco dell'impiccato con le seguenti caratteristiche:

-un sistema di ranking gestito tramite username chiesto a fine partita
-multi-lingua (ita-eng)
-tre livelli di difficoltà (facile, medio, difficile) differenziati dal numero di lettere della singola parola e dalla grandezza del pool di parole
-implementazione di un'interfaccia grafica (opzionale)
